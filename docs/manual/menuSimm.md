# Menu menuSimm

This menu is used for Simulation Mode (SIMM) fields of input record types that
can fetch either raw or engineering values through their SIOL link.

| Index | Identifier | Choice String |
| ----- | ---------- | ------------- |
| 0 | menuSimmNO | NO |
| 1 | menuSimmYES | YES |
| 2 | menuSimmRAW | RAW |
